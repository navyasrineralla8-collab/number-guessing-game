# 🎮 Number Guessing Game in Python

This is a simple number guessing game built using Python.

## 🚀 How it works
- The computer randomly selects a number between 1 and 100
- The user tries to guess the number
- The program gives hints:
  - Too high 📈
  - Too low 📉
- The game ends when the correct number is guessed
- it have 5 chances to guess the game if the person did't guess the currect number in 5 chances the person will lose the game
  

## 🛠️ Features
- Random number generation
- User input handling
- Hint system (high/low)
- Tracks number of attempts
- Error handling for invalid input

## ▶️ How to Run

1. Make sure Python is installed
2. Run the following command:

```bash
python guess.py
